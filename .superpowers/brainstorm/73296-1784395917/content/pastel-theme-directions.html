C. Toy Table

Uses one random pastel as the base but allows a warm neutral secondary surface. Friendlier, but less strictly “entire UI from one color.”

Runtime behaviorPick a base pastel once on dev server start. The browser receives that seed and does not reshuffle on refresh.
Color relationshipButtons are lighter than the base; selected states and dark surfaces are deeper relatives of the base.
FontUse a rounded system stack so it feels softer without depending on any external font download.
